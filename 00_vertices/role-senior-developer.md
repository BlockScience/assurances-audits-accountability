# Senior Developer Role

## Identity

The Senior Developer role is a development position responsible for feature design, code review, and mentoring junior developers.

## Capabilities

A Senior Developer can:
- Design and implement complex features
- Review and approve code changes
- Mentor junior developers
- Make technical decisions within scope

## Constraints

- Must follow team coding standards
- Cannot approve own code
- Must coordinate deployments with Platform Team
