# Developer Role

## Identity

The Developer role is a development position responsible for implementing features, writing tests, and fixing bugs.

## Capabilities

A Developer can:
- Implement features from specifications
- Write unit and integration tests
- Fix bugs and issues
- Participate in code reviews

## Constraints

- Must follow team coding standards
- Requires review for all changes
- Must pass all tests before merging
