# Deploy to Production

## Definition

The responsibility for executing and approving deployments to production environments.

## Scope

- Executing deployment pipelines
- Approving deployment requests
- Verifying successful deployment
- Rolling back if needed

## Constraints

- Requires approval from accountable party
- Must occur during change windows (except emergencies)
- Must be logged and auditable
