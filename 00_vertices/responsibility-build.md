# Build Application

## Definition

The responsibility for building application code and creating deployment packages.

## Scope

- Compiling/building application code
- Running automated tests
- Creating deployment packages
- Validating build artifacts

## Constraints

- All tests must pass
- Code must be reviewed
- Artifacts must be versioned
